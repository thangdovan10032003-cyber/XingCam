# assets/overlays/

Place your light-leak and grain PNG overlay images here.

## Required files (referenced in code)

| File                    | Description                                 |
|-------------------------|---------------------------------------------|
| `light_leak_1.png`      | Warm orange/yellow light leak (top-right)   |
| `light_leak_2.png`      | Cool blue lens flare (left edge)            |
| `light_leak_3.png`      | Magenta/pink double-exposure bleed          |
| `grain_texture.png`     | Static high-res grain texture (fallback)    |

## Where to get free overlays

- **Unsplash / Pexels** — search "light leak overlay PNG transparent"
- **Textures.com** — grain textures
- **RetroSupply Co.** — paid, high quality film overlays
- **DIY** — photograph a blank card with a light source and key out the black in Photoshop/GIMP

## Usage tip

Images should be:
- PNG with transparency (or pure black background for BlendMode.screen)
- At least 1080×1920 px so they cover all screen sizes
- Exported at 72 dpi for optimal file size
