# XingCam: Sovereign Creative Ecosystem

XingCam is a professional-grade, privacy-first photography suite built with Flutter. It combines a high-performance Retro Camera engine with a state-of-the-art AI Studio, all operating within a strictly sovereign, offline-first architecture.

## 🛡️ Sovereign Principles
- **100% Zero-Telemetry**: No tracking, no data leaks, no external dependencies for core creative work.
- **Offline-First AI**: All 33+ AI tools operate on-device or via secure local proxies.
- **Project-Wide Tokenization**: 100% design system purity using semantic tokens.
- **Global Localization**: Full support for English and Vietnamese with context-aware pluralization.

## 📸 Core Features
- **Retro Camera Engine**: 4K capture with professional 3D LUT shaders, real-time light leaks, and analog film grain.
- **AI Studio**: 33+ standardized tools including Sky Replacement, Object Removal, and Portrait Relighting.
- **Sensory UI**: Haptic-rich controls, sliding panels, and aesthetic "Zine" layouts.
- **Sovereign Gallery**: Multi-select management with AI-driven "Best Shot" evaluation.

## 🛠️ Technical Stack
- **Framework**: Flutter (Dart)
- **State Management**: BLoC / Cubit
- **I18n**: Easy Localization
- **Design System**: XingDesign Tokens (AppColors, AppHeader, AppSlider)
- **Rendering**: Custom GLSL Shaders & LUT Engine

## 🚀 Getting Started
1. Run `flutter pub get`
2. Run `dart run build_runner build --delete-conflicting-outputs`
3. Launch on iOS/Android
